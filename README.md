Super Store Sales Analysis (Excel + Power BI)

End-to-end analytics project using **Excel (Power Query)** to transform raw data into a structured model, then **Power BI** to build an interactive dashboard.

Project Goal
Turn a single raw Super Store table into an analysis-ready dataset and deliver a dashboard that answers:
- How do Sales and Profit perform over time?
- Which segments and categories drive the most value?
- What are the most profitable products?

Repository Structure
- Excel/  
  Excel workbook + Power Query transformations + clean tables (Fact/Dimensions)

- PowerBI/ 
  Power BI dashboard (.pbix) and a screenshot of the report (optional)

Key Work Performed
 1) Data Preparation (Excel / Power Query)
- Cleaned and standardized data types
- Created a `Delivery days` feature
- Built a simple data model:
  - Fact_orders
  - Dim_Customer
  - Dim_Product

2) Reporting (Power BI)
- Designed a dashboard with KPIs and breakdowns such as:
  - Total Sales / Total Profit / Order Count
  - Sales by Segment and Category
  - Profit by Category
  - Monthly Sales trend
  - Top profitable products
  - Year and Country filters

Tools
- Excel (Power Query)
- Power BI

How to Use
1. Open the Excel file in Excel/ if you want to review the transformations and tables.
2. Open the Power BI file in PowerBI/ to explore the dashboard interactively.
