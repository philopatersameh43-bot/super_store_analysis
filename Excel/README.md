 Excel Analysis – Super Store Sales

 Overview
This folder contains the Excel-based data preparation and analysis of the Super Store dataset.

The original dataset was provided as a single large table. Using Power Query, the data was transformed, cleaned, and structured into multiple logical tables before building the Power BI dashboard.

 Data Preparation (Power Query in Excel)

Using Power Query, the following steps were performed:

- Cleaned and formatted raw data
- Removed unnecessary columns
- Ensured correct data types
- Created structured tables from one large dataset
- Separated data into logical tables such as:
  - Orders
  - Customers
  - Products
  - Fact Orders

This improved data organization and prepared the dataset for analysis.

 Analysis Performed in Excel

- Created structured tables
- Built Pivot Tables
- Calculated key metrics:
  - Total Sales
  - Total Profit
  - Sales by Segment
  - Profit by Category
  - Monthly Sales Trends

 Key Insights Identified

- Consumer segment generated the highest sales.
- Technology category produced the highest profit.
- Sales show stronger performance toward the end of the year.

 File Included

- `Supert_store.xlsx` – Excel file containing Power Query transformations, structured tables, and Pivot Table analysis.
